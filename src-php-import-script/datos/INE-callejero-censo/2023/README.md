
Para obtener el archivo "caj_esp_012023.zip" con el fichero nacional del censo electoral para poder casar la información de los códigos postales con cada municipio:

**Página web del INE**

Recurso: [Cartografía secciones censales y callejero de Censo Electoral](https://www.ine.es/ss/Satellite?L=es_ES&c=Page&cid=1259952026632&p=1259952026632&pagename=ProductosYServicios%2FPYSLayout)

Hay dos secciones:

- Cartografía digitalizada de secciones censales
- Callejero de Censo Electoral
  
De la segunda sección: **Callejero de Censo Electoral**, elegimos **2023 Enero** en el selector de **"Ficheros nacionales"** y pulsamos el botón [Ir] para descargar el archivo zip: **caj_esp_012023.zip**

Dentro del ZIP vienen varios archivos y los que nos interesan son:

- TRAM.P01-52.D221231.G230306 lo extraemos y renombramos a INE-CENSO-TRAMOS-NAL.txt (1.491.105 líneas)
- VIAS.P01-52.D221231.G230306 lo extramos y renombramos a INE-VIAS-NAL.txt (897.630 líneas)
   
Podemos descargar el documento con el ["Diseño de los registros de los ficheros"](https://www.ine.es/ss/Satellite?blobcol=urldata&blobheader=application%2Fmsword&blobheadername1=Content-Disposition&blobheadervalue1=attachment%3B+filename%3Ddisenocallejero.doc&blobkey=urldata&blobtable=MungoBlobs&blobwhere=817%2F26%2Fdisenocallejero%2C1.doc&ssbinary=true) que se recoge en el extracto de la orden de 11 de julio de 1997
