
**CENTRO NACIONAL DE INFORMACIÓN GEOGRÁFICA (CNIG) dependiente del Ministerio de Transportes**

Los datos del CNIG suelen incluir el CÓDIGO INE con cinco cifras cuando se corresponden al nivel de “Municipio” del Instituto Nacional de Estadística, es decir el Código de Provincia (2 cifras) + el código de municipio (3 cifras), por lo que disponer de nuestra base de datos de provincias y municipios españoles usando los identificadores del INE nos permitirá vincular fácilmente estos datos del CNIG.

Vamos al ["Centro de Descargas" del CNIG](http://centrodedescargas.cnig.es/CentroDescargas/index.jsp) para acceder a la Información Geográfica de Referencia (Datos topográficos básicos necesarios para la representación del territorio).

**1.** Descargamos el "Nomenclátor Geográfico de Municipios y Entidades de Población" pulsando el botón "Descargar" bajo el título mencionado.

Obtendremos el archivo "BD_Municipios-Entidades.zip", del que nos interesa el siguiente:

- MUNICIPIOS.csv (8.132 líneas) que extraemos y renombramos como MUNICIPIOS.txt 

De aquí podremos sacar, de cada municipio, su posición geográfica (expresada en centroide, latitud, longitud), la población de cada municipio, superficie en m2, capital de provincia, etc.

**2.** Descargamos el "Nomenclátor Geográfico Básico de España" pulsando el botón "Descargar" bajo el título mencionado.

Obtendremos el archivo "NGBE.zip", del que nos interesan los siguientes archivos:

- El archivo en formato CSV llamado "ngbe.csv" (1.165.258 líneas)

- El archivo PDF contiene -entre otra literatura- la descripción de los campos de la base de datos de este Nomenclátor Geográfico Básico para poder interpretarlos.

Este listado nos permitiría disponer de datos (incluyendo geoposición) más allá de los municipios, por ejemplo: puertos/aeropuertos, comarcas, barrios, entidades menores de población, playas, embalses, montañas, edificaciones, etc.
